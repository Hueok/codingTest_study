## 풀이 : #itertools, #단순 Implementation
> 이 문제는 딱히 풀이할 게 없다. itertools를 이용하면 단순히 구현이 가능하다.

## Code
```py
import itertools

def print_cases(query):
    for s in itertools.combinations(query[1:], 6):
        print(' '.join(list(s)))

while True:
    query = list(input().split())
    if query == ['0']:
        break
    print_cases(query)
    print()

```
