## 풀이 : DFS

### check func  
```
두 수의 부등호 성립 관계 확인 -> boolean return
```

### dfs func  
```
깊이가 K+1에 도달하면 return.
i의 range를 0~9까지 돌리면서 방문 여부 확인. -> 방문 시 스킵
깊이가 0이면 방문처리 후 dfs(깊이+1)
깊이가 1 이상이면서 return of check()이 True 1이면 방문 처리 후 dfs(깊이+1)
루프가 range 범위를 넘으면 no return -> recursion 빠져나와 visited를 미방문 처리
```

## Code
```py
K = int(input())
signs = input().split()
visited = [0]*10
result =[]

def check(x, y, s):
    if s == '<':
        return x<y
    if s == '>':
        return x>y

def dfs(cnt, num):
    if cnt == K+1:
        result.append(num)
        return
    
    for i in range(10):
        if visited[i]:
            continue
        if cnt == 0 or check(num[cnt-1], str(i), signs[cnt-1]):
            visited[i] = 1
            dfs(cnt+1, num+str(i))
            visited[i]=0

dfs(0, '')
result.sort()
print(result[-1])
print(result[0])
```
